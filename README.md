# fine-tuning-kogpt2

## Introduction
SKT가 공개한 KoGPT2 모델을 한국어 노래가사 20곡을 이용해 파인튜닝하였습니다. 적은 데이터로도 가사를 꽤 잘 생성합니다. 더 큰 데이터를 사용하면 성능도 더 좋아질 것으로 보입니다.

## Requirements
```
Windows 10 Pro
Python 3.8.16
Torch 1.8.0
CUDA 11.1
cuDnn 8.5.0
```
